## 🚀 Funcionalidades

* **Ajuste Dinâmico:** Controle total sobre a duração dos dias.
* **Sincronização:** Compatível com servidores multiplayer.
* **Performance:** Código otimizado para não causar TPS drop.

## 🛠️ Instalação

1.  Certifique-se de ter o **Forge** ou **Fabric** instalado.
2.  Baixe o arquivo `.jar` na aba de [Releases](#).
3.  Coloque o arquivo na pasta `mods` do seu diretório `.minecraft`.
4.  Inicie o jogo e divirta-se!

## 🔧 Configuração

O arquivo de configuração pode ser encontrado em `.minecraft/config/ldnf.json`.

| Parâmetro | Tipo | Descrição |
| :--- | :--- | :--- |
| `dayLength` | Inteiro | Define a duração do dia em ticks. |
| `nightLength` | Inteiro | Define a duração da noite em ticks. |
| `smoothTransition` | Boolean | Ativa transições suaves de iluminação. |
